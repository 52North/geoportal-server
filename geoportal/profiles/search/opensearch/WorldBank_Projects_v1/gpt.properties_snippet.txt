# Worldbank - Projects
catalog.search.searchSite.worldbanksearch          = World Bank Projects
catalog.search.searchSite.worldbanksearch.abstract = World Bank Projects Portfolio provides access to basic information on all of the World Bank's lending projects from 1947 to the present. The dataset includes basic information such as the project title, task manager, country, project id, sector, themes, commitment amount, product line, and financing.
